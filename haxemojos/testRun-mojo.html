haxemojos:testRun

Full name:

com.tenderowls.opensource:haxemojos-maven-plugin:1.0-SNAPSHOT:testRun

Description:

Run tests with `neko`.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: test.

Optional Parameters

Name Type Since Description
defines List - Define a conditional compilation flag
resources List - Add a named resource files 
<resources>
    <resource>
        <name>license-test</name>
        <file>LICENSE.txt</file>
    </resource>
</resources>

verbose boolean - Turn on verbose mode
