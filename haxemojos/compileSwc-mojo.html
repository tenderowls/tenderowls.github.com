haxemojos:compileSwc

Full name:

com.tenderowls.opensource:haxemojos-maven-plugin:1.0.0-RC2:compileSwc

Description:

Compile SWC for Adobe(R) Flash Player(TM) or Adobe(R) AIR(TM).

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: compile.
  • Binds by default to the lifecycle phase: compile.

Required Parameters

Name Type Since Description
main String - Main class
swfVersion String - Change the SWF version (6 to 11.x)
Default value is: 11.2.

Optional Parameters

Name Type Since Description
debug boolean - Compile in debug mode
defines List - Define a conditional compilation flag
flashStrict boolean - More type strict flash API
flashUseStage boolean - Place objects found on the stage of the SWF lib
resources List - Add a named resource files 
<resources>
    <resource>
        <name>license-test</name>
        <file>LICENSE.txt</file>
    </resource>
</resources>

swfHeader String - Define SWF header (width:height:fps:color)
verbose boolean - Turn on verbose mode
